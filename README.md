# product-landing-lusineh# LumaSip — Product Landing Page

A responsive product showcase landing page built as part of a GitHub branching and pull request workflow assignment.

## Features

### Landing Page v1
- Hero section with product name, tagline, and CTA
- Product showcase with feature cards
- Benefits section with icons
- Testimonials section
- Pricing section with multiple tiers
- Responsive layout using semantic HTML and CSS

### Merged Enhancements
- Sticky navigation bar with section links
- Smooth scrolling for anchor links
- Active section highlighting on scroll
- CTA modal with form validation and multiple close behaviors

> Additional gallery interactions were explored in a separate feature branch and kept for a future iteration.

## How to Run Locally
No build step is required.

1. Clone the repository
2. Open `index.html` in your browser
